Arch Linux kurulumu

Geliştirici BerkeMayıs 9, 2021Mayıs 9, 2021Arch Linux kurulumu için bir yorum yapın

Arch, tty ekranından kurulumu yapılan bir dağıtımdır. Calameres kurucusunun yaptıklarını terminalden yapıyoruz aslında. Arayüz ile kurulum yapacaksanız archinstall komutunu da kullanabilirsiniz.

Disk bölümleme

Disk bölümlemek için fdisk, gdisk, cfdisk gibi araçlar kullanabilirsiniz. Ben kullanıcı arayüzü olduğu için cfdisk tavsiye ediyorum. MBR diskler için sadece kök bölümü oluşturmanız yeterli. UEFI kurulumda EFI ile kök bölümünü ayırmanız gerek. Kuruluma başlar başlamaz loadkeys trq komutuyla klavyenizi ayarlayın.

Bölümlendirme tablosu oluşturma

parted /dev/sdX

Komutunu girerek kurulum yapacağınız diskte parted açın. Parted açıldıktan sonra mklabel msdos komutuyla MBR yazdırın ya da UEFI kurulum yapacaksanız gpt yazdırın.

Diskte bölüm açma

cfdisk /dev/sdX komutunu girip cfdiski açın. Cfdisk açıldıktan sonra UEFI kurulum yapacaksanız 2 tane, Legacy kurulum yapacaksanız 1 tane bölüm açın. Write butonuna basıp çıkın.

Bölümlere dosya sistemi yazdırma

mkfs komutuyla bölüm üzerine dosya sistemi yazdırabilirsiniz. Kök bölümüne mkfs.ext4 /dev/sdXX komutuyla ext4 dosya sistemini yazdırın. UEFI kurulum yapıyorsanız mkfs.fat -F32 komutuyla FAT32 dosya sistemini yazdırın.

Bölümleri bağlama

# mount /dev/sd<kök bölümü> /mnt
# mkdir /mnt/boot
# mount /dev/<EFI bölümü> /mnt/boot

İnternete bağlanma

iwctl aracını kullanarak internete bağlanabilirsiniz. iwctl komutunu girip iwctli açın. Açıldıktan sonra station list komutuyla ağ bağdaştırıcılarını listeleyin. Mesela ağ bağdaştırıcınız wlan0 ise station wlan0 connect <SSID> komutuyla ağınıza bağlanabilirsiniz. Bağlandıktan sonra CTRL+C kombinasyonuyla çıkın. Kablolu bağlantılarda otomatik olarak bağlanır.

Ana sistemin kurulumu 

pacstrap /mnt base linux linux-firmware nano

komutuyla sistemi kurun. Kurulduktan sonra chroot ile o sisteme geçin. 

genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt

Chroota geçtiğinizde shell promptunun değiştiğini göreceksiniz.

Zaman diliminin ayarlanması

# ln -sf /usr/share/zoneinfo/Europe/Istanbul /etc/localtime
# hwclock --systohc

Dilin ayarlanması

Dil olarak İngilizce seçmenizi tavsiye ederim. nano /etc/locale.gen komutuyla locale dosyasını açın ve en_US.UTF-8 UTF-8 satırının başındaki # işaretini kaldırın. Ardından locale-gen komutunu girin. 

nano /etc/locale.conf

komutuyla locale.conf dosyasını açıp içine LANG=en_US.UTF-8 yazıp kaydedip çıkın.

Ağ konfigürasyonu

nano /etc/hostname komutuyla hostname dosyasını açın. Dosyanın içine kullanıcı adınızı yazın. Ardınan nano /etc/hosts komutuyla hosts dosyasını açıp içine bunu yapıştırın. kullaniciadi yazan kısmı kullanıcı adınızla değiştirin.

127.0.0.1	localhost
::1		localhost
127.0.1.1	kullaniciadi.localdomain	kullaniciadi

# pacman -Sy networkmanager
# systemctl enable NetworkManager

Kullanıcı oluşturma

useradd komutuyla kullanıcı oluşturabilirsiniz. Bu komutu bu şekilde kullanacaksınız: 

useradd -m -g users -G wheel,storage,power -s /bin/bash kullanıcıadi

Şifresini belirlemek için: 

# passwd kullaniciadi
# passwd //Kök kullanıcı şifresi için.

Kullanıcıya sudo kullanma yetkisi verme

Önce pacman -Sy sudo komutuyla sudoyu kurun. Kurduktan sonra sudoers dosyasını nano ile açın: 

EDITOR=nano visudo
Açıldıktan sonra "%wheel ALL=(ALL) ALL" satırının başındaki # işaretini kaldırın ve dosyanın sonuna "Defaults rootpw" yazın.

Önyükleyici kurma

Önyükleyici olarak GRUB kuracağım. GRUB kurarken eğer Legacy sisteminize kuruyorsanız bu komutları girmelisiniz: 

# pacman -S grub efibootmgr
# grub-install /dev/sdX

UEFI sisteme kurarken bu komutları girmelisiniz: 

# pacman -S grub efibootmgr
# grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB

GRUB config dosyası oluşturma

grub-mkconfig -o /boot/grub/grub.cfg

Yeniden başlatma

Bu adımlardan sonra temel sistem kurulmuş olacaktır. Yeniden başlatıp kurulumun sorunsuz bitip bitmediğine bakalım. Sorunsuz bittiyse sizi tty ekranı karşılayacaktır.

# exit 
# umount -R /mnt
# reboot

Masaüstü ortamı/Pencere yöneticisi kurulumu

pacman -S xorg-server xorg-apps xorg-xinit xorg-twm xorg-xclock

komutuyla Xorg’u kurun. İnternete bağlı değilseniz nmtui komutuyla bağlanın.

i3 kurulumu: sudo pacman -Syu i3-wm komutu i3 ile ilgili gerekli paketleri kurar. Pencere yöneticileri için giriş yöneticisi olarak ly kullanmanızı tavsiye ederim.

bspwm kurulumu: sudo pacman -Syu bspwm

MATE kurulumu: sudo pacman -Sy pacman mate mate-extra lightdm

Cinnamon kurulumu: sudo pacman -Sy cinnamon gnome-terminal lightdm

KDE kurulumu: sudo pacman -Syu plasma dolphin konsole sddm

GNOME kurulumu: sudo pacman -Syu gnome gdm

Giriş yöneticilerini aktif etme

Aşağıdaki komutlara göre indirdiğiniz pencere yöneticisini aktif edebilirsiniz. 

# sudo systemctl enable lightdm
# sudo systemctl enable gdm
# sudo systemctl enable sddm

Ve soununda Arch kurulmuş olacak.
